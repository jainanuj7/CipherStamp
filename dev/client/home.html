Overview



File Upload

You create a digital fingerprint of your file on your own computer. The file whose integrity is to be maintained can be of any format such as JPEG, PDF, DOCX, XLSX, MP3, MP4.

Block Generation

Hash of the file is calculated and embedded on a tamperproof and unhackable ledger along with the geolocation of the user and timestamp.

Verification

Integrity of a file can be checked in the future simply by uploading it again. Even a slight change in the file will be detected.

Retrieval

Using the secret key, you can retrieve the original file anywhere, anytime. The timestamp and geolocation can also be fetched for future references.

Timestamping & Blockchain - a powerful combination

CipherStamp leverages blockchain technology to create next-level geolocation based timestamps

Worldwide Acceptance

CipherStamp operates fully independent from centralized organizations, governmental authorities or consortiums.

Unlimited Validity

Once a hash fingerprint is embedded in the blockchain, it is immutable and will exist "forever" as a trusted timestamp.

Perfect Confidentiality

CipherStamp keeps all files confidential using AES-256 end-to-end encryption. Files can be only retrieved using user's secret key.

Free of Charge

All of our services are free of cost!

Transparent & Immutable

CipherStamp uses a decentralized distributed ledger. Your files can't be altered retroactively without alteration of all subsequent blocks.

Location Based Timestamping

Along with timestamp of the file upload, CipherStamp also keeps a track of the geolocation. This ensures that the origin of the file is recorded efficiently.

Designed with developers in mind

CipherStamp API is designed to give developers a fast, reliable and scalable tool to create secure location based timestamps. Developers can use our highly extesible APIs to build their own web or mobile based applications to verify the integrity of existing digital files in their system.

Our service is open source and can be extended according to developer's requirements. Fork us on GitHub!

Frequently Asked Questions

Find an answer to your question

What is a blockchain-based timestamp?

A blockchain-based timestamp is a timestamp that is stored by thousands of computers distributed around the globe. This property makes blockchain-based timestamps extremely secure and independent of a trusted third party.

Can anybody else access my file?

No. The content of your files are stored in encrypted format. It can only be accessed by your secret key.

How is the timestamp calculated?

The timestamp represents date and time when a cryptographic fingerprint (hash) of your submitted file was included in the blockchain.

Aren't you another trusted third party?

Absolutely not. You can verify a timestamp independently and without relying on us.

Is it mandatory to disclose my location?

No. It is an optional feature to enhance file security.

Do I need to pay for your services?

No. All of our services are free of cost.
